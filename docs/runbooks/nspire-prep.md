# Runbook: NSPIRE Pre-Inspection Self-Audit

## Purpose
Walk the property with this checklist before scheduling the official NSPIRE inspection. Identify and remediate failures before the PHA inspector arrives.

## How to Run

1. Open Dashboard → Inspections
2. Select the deal
3. Click "Generate Checklist"
4. Walk each area of the property, marking pass/fail/n/a
5. Review preFailAudit output for critical items
6. Remediate all "fail" items
7. Regenerate checklist after fixes

## Checklist by System

### Site
- [ ] No erosion, sinkholes, or unstable ground
- [ ] Walkways clear and safe (no trip hazards, ice removal in winter)
- [ ] Fencing intact where required
- [ ] Exterior lighting functional
- [ ] Trash disposal adequate
- [ ] No standing water or drainage issues

### Building Exterior
- [ ] No foundation cracks or structural damage
- [ ] Roof intact (no missing shingles, no visible leaks)
- [ ] All windows intact (no broken glass)
- [ ] Exterior doors lock and close properly
- [ ] No peeling paint on exterior surfaces (pre-1978: lead protocol)
- [ ] Stairs, railings, and balconies secure
- [ ] Gutters and downspouts functional
- [ ] Soffit and fascia intact

### Building Systems
- [ ] HVAC functional in all zones
- [ ] No visible plumbing leaks
- [ ] Electrical panel: no exposed wiring, no double-taps, proper breakers
- [ ] Elevator operational (if applicable)
- [ ] Fire suppression system operational (if applicable)
- [ ] Smoke detector in every bedroom, hallway, and common area
- [ ] CO detector on every floor
- [ ] Main electrical panel clean
- [ ] Water heater: functional, T&P valve present

### Common Areas
- [ ] No mold or moisture in common areas
- [ ] No pest infestation
- [ ] Adequate lighting
- [ ] Egress not blocked
- [ ] Laundry equipment functional (if applicable)
- [ ] No fire hazards in storage areas

### Each Unit
- [ ] Entry door lock functional
- [ ] All windows open, close, and lock
- [ ] Heating system functional
- [ ] All plumbing functional (hot water, no leaks)
- [ ] Electrical outlets and switches functional
- [ ] Smoke detector present and working
- [ ] CO detector present and working
- [ ] No mold or moisture intrusion
- [ ] No pest infestation
- [ ] No peeling/chipping paint (pre-1978: lead protocol)
- [ ] Refrigerator functional
- [ ] Stove/range functional
- [ ] Bathroom fan functional
- [ ] No trip hazards (floor, stairs)
- [ ] Dryer exhaust vent connected and clear

## Scoring

NSPIRE score ≥ 60 required to pass. Target 80+.

| Severity | Deduction |
|---|---|
| Life-threatening | -15 pts |
| Severe | -8 pts |
| Moderate | -4 pts |
| Low | -1 pt |

## Common Quick Wins (low cost, high impact)

1. **Smoke/CO detectors** — $15–30 each. Instant removal of life-threatening deductions.
2. **Window screens** — $20–50 each.
3. **Door hardware** — $20–100. Entry door locks must be functional.
4. **Lighting** — Replace burnt bulbs. $5–50.
5. **Clean pests** — Exterminator call-out. $150–300.

## When to Call a Pro

- Electrical panel issues → licensed electrician
- Roof damage → roofing contractor
- Lead paint → certified abatement contractor
- Mold > 10 sq ft → licensed remediation
- Foundation cracks → structural engineer assessment
